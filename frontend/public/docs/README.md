# AegisAI – Diploma documents

Drop your diploma artefacts here so they are served by the website.

Recognised files:

| File name                | Purpose                                  | Linked from           |
| -----------------------  | ---------------------------------------- | --------------------- |
| `AegisAI_Diploma.docx`   | Main Word diploma document.              | `/documentation` page |
| `AegisAI_Diploma.pdf`    | (Optional) PDF export of the diploma.    | `/documentation` page |
| `AegisAI_README.pdf`     | (Optional) PDF export of the README.     | `/documentation` page |

If a file is missing, the Documentation page will gracefully show a
"will be available after adding …" notice instead of breaking.
